Como este site foi feito?

Neste site estamos vendo alguns testes usandos HTML5 e CSS3, duas línguagens, mas não de programação, e sim de marcação, caso você não saiba o que significa a sigla HTML e CSS, arraste o mouse por cima das duas primeiras siglas e descubra o significado das duas, elas fazem parte de um grupo chamado front-end, desenvolvedores front-end são pessoas que desenvolvem com: HTML5, CSS3 e JavaScript

Para quê serve a HyperText Markup Language e as Cascading Style Sheets

A HTML, é usada para a criação do conteúdo de um site, isto é, criar isso aqui que você está lendo, além de ser uma das bases iniciais para iniciar a construção de um site, a atual versão da HTML é a versão 5, por isso, é HTML5. Ás CSS, é usada para fazer a forma do site, o design dele, mas isto não quer dizer que você obrigatoriamente precisa ser designer, não, são funções diferentes, você só vai receber a paleta de cores, fontes e o resto do estilo do site que foi desenhado pelos designers e aplicar elas no seu site, mas é importante saber um pouco disso enquanto aprende CSS que está na versão 3, por isso, é CSS3.

Como usar uma Favicon?

Para usarmos uma Favicon em nosso site, primeiro precisamos ter uma, claro, existem alguns sites como o Iconarchive ou converter uma favicon nossa que foi
feita em png para .ico usando o favicon.io, ou até mesmo desenhar a sua usando o favicon.cc. Após isso, criaremos uma pasta para o nosso site, colocamos
a imagem já em .ico e abriremos a pasta com o Visual Studio Code, criaremos um arquivo html5.html ou index.html, usamos "!" para criar o código base, editamos
nosso idioma e etc, depois disso tudo vamos para <head>, lá, acima da linha do título, digitamos link, e vamos até "link:favicon" onde ao apertar, criará um código,
e dentro dele, vamos em "rel", apagamos o favicon padrão do código e usamos: CTRL + Spacebar, onde podemos escolher a imagem que está em .ico.

Tipos de target e rel

  1. Usando o parâmetro target="_blank", abrirá uma nova janela no google
  2. Usando o parâmetro target="_self", Vai abrir o link na janela atual (Padrão)
  3. Usando o parâmetro target="_top", Vai desfazer todos os frames e abrir no navegador completo
  4. Usando o parâmetro target="_parent", Similiar ao top, em referência à janela mãe
  5. Usando o parâmetro target="Nome-do-frame", Caso esteja usando frames, indicar o nome da janela á abrir
    • Usando o parâmetro rel="external", indicará que o link é para outro site que não faz parte do atual
    • Usando o parâmetro rel="next", indica que o link é para a próxima parte do documento atual
    • Usando o parâmetro rel="prev", indica que é parte anterior do atual
    • Usando o parâmetro rel="author", indica que é um link para um site do autor do artigo atual
    • Usando o parâmetro rel="nofollow", isso funciona pro mecanismo de busca, você está dizendo pra ele que não um avalo para esse conteúdo, pode ser o link de uma parceria, de uma matéria que você não sabe se é real/oficial, mas, existe cliente que não quer link nofollow.
